# 🍎 Apple — 광장 AI 바텐더 (배포 방법)

파인앤코 광장의 AI NPC "Apple"을 돌리는 Cloudflare Worker입니다.
**API 키 없음** (Workers AI 바인딩이라 키가 아예 필요 없어요). **무료 하루 10,000 뉴런 = 약 900회 대화.**

---

## 배포 (3단계, 5분)

터미널에서 이 폴더(`plaza-ai`)로 들어간 다음:

### 1. Cloudflare 로그인
```
npx wrangler login
```
브라우저가 열리면 붐 Cloudflare 계정으로 허용 누르면 됩니다.

### 2. 배포
```
npx wrangler deploy
```

### 3. 주소 복사
배포가 끝나면 이런 주소가 나와요:
```
https://plaza-ai.<붐계정이름>.workers.dev
```
**이 주소를 저에게 알려주시면** 게임에 연결해 드릴게요.
(또는 게임 파일에서 `APPLE_AI_URL` 을 찾아 그 주소로 바꾸면 바로 됩니다.)

---

## 지금 안 해도 됩니다

배포 전에도 **광장의 Apple은 작동해요.** 서버가 없으면 게임이 자동으로
**로컬 추천 모드**(진짜 메뉴 데이터로 규칙 기반 추천)로 돌아갑니다.
배포하면 그때부터 자유로운 대화가 되는 거예요.

---

## 돈이 나가나요?

**안 나가요.**
- Workers Free 플랜은 하루 10,000 뉴런까지 무료, **넘으면 과금이 아니라 에러**입니다.
- 결제수단을 등록하지 않으면 구조적으로 청구가 불가능해요.
- 한도를 넘으면 게임이 자동으로 로컬 추천 모드로 폴백하고, 다음날 00:00 UTC(한국시간 오전 9시)에 리셋됩니다.

도배 방지도 넣어놨어요: **IP당 5초에 1번, 시간당 30번.**

---

## 메뉴가 바뀌면

`src/index.js` 안의 `MENU` 상수를 고치고 다시 `npx wrangler deploy` 하세요.
AI는 파인앤코를 모르기 때문에, **여기 적힌 메뉴만** 근거로 답합니다.
(가격·재료를 지어내지 않게 하는 안전장치예요)

## 모델 바꾸기

`src/index.js` 맨 위 `MODEL` 을 바꾸면 됩니다.
- `@cf/meta/llama-3.1-8b-instruct` — 현재값. 대화당 ~10뉴런 → **하루 ~900회**
- `@cf/meta/llama-3.3-70b-instruct-fp8-fast` — 더 똑똑하고 한국어 좋음. 대화당 ~65뉴런 → **하루 ~150회**
