Pine & Co — Cocktail Defense  ·  업데이트 업로드
================================================

이번 업데이트는 새 이미지 1개(노부킹 보스)가 있어서
index.html + 이미지(assets/boss_nobook_stand.png) 를 같이 올려야 합니다.
총 3개 파일 (100개 이하).

[들어있는 것]
  index.html                      ← 게임 본체(최신)
  sandbox.html                    ← 테스트 모드(안 바뀜, 처음이면 같이)
  assets/boss_nobook_stand.png    ← ★ 새 보스 이미지 (노부킹 무리). 이게 빠지면 보스가 그림 없이 네모로만 나옴.

[이번 업데이트 내용]
  1) 노부킹 무리(가로 벽 보스) 추가  ← 새 이미지 필요
     - 초반(5번째 보스 전)에만, 한 줄로 늘어선 무리가 화면을 천천히 횡단.
     - 체력 높고 접촉 데미지 쎔. 좌우로 못 피하게 막음.
  2) 스피릿(칵테일 기주) 드랍 아주 희박하게
  3) 하이스코어 시즌 리셋 (로컬 기록 자동 비움)

[올리는 법 — GitHub 웹]
  1) 저장소 → "Add file" → "Upload files"
  2) 이 폴더의 index.html (+ 처음이면 sandbox.html) 을 드래그 → 기존 거 덮어쓰기
  3) 같은 화면에서 assets 폴더(안에 boss_nobook_stand.png)도 같이 드래그
     → 기존 assets/ 에 이미지 1개가 추가됨 (기존 이미지는 안 지워짐)
  4) Commit changes

  * 전체를 새로 올리고 싶으면 'github-업로드' 폴더(전체)를 쓰세요.
    이 폴더는 "이번에 바뀐 것만" 모아둔 겁니다.

[공유 순위 비우기 — 구글 시트에서 직접 (1분)]  ★ 하이스코어 진짜 리셋은 이 단계
  코드만 올려선 공유 랭킹(모두가 보는 순위)은 안 지워집니다. 시트를 비워야 합니다.
  1) 랭킹이 저장되는 구글 시트 열기
  2) 아래쪽 탭 'scores' 선택
  3) 2행부터 끝까지 전부 삭제 (1행 머리글 time_iso|name|time_sec|kills|sales 는 남김)
  4) 저장. 이러면 모두의 공유 기록이 0부터 다시 시작. (Apps Script 재배포 불필요)

[폰에서 확인]
  배포 후 안 바뀌어 보이면 새로고침, 또는 Safari 설정 → 방문기록·데이터 삭제 한 번.
